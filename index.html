DEDCPU-16

D emulator for DCPU-16

Download this project as a .zip file Download this project as a tar.gz file

D DISassembler for DCPU-16 v1.7

Usage:

./ddis filename [options]

Prints in stdout a disassembled code of input file. It can add coments to the generated code showing what hex words and what addres generated each instrucction. In addtion can auto-label jumps, making more easy to read and understand disassembled code.
